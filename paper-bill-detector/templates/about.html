About the Website

Pa Barya is a computer vision–based web application designed to detect paper bills from uploaded images and compute their total monetary value. After detecting the bills, the system calculates the exact coin change based on the coin denominations selected by the user.

This project demonstrates the practical application of object detection using deep learning, combined with a modern web-based interface for real-world usability.
